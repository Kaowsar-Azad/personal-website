A modern and responsive personal portfolio website showcasing services, projects, and professional information with a clean and minimal design.

## 📌 Project Overview

This is a personal portfolio website designed to present personal information, services, pricing plans, and blog sections in a clean and professional layout. The website focuses on user-friendly design and responsiveness.

---

## ✨ Features

- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- 🎨 Clean and minimal UI
- 👤 Personal introduction section
- 🧾 About section
- 🛠️ Services section
- 💰 Pricing plans section
- 🖼️ Portfolio/gallery section
- 📰 Blog section
- 📩 Contact section (if added)
